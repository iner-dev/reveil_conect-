Contribuer au projet "Réveil Connecté"
=====================================

Merci de votre intérêt pour contribuer ! Ce document décrit brièvement le flux de contribution attendu.

1) Issues
- Ouvrez une issue décrivant le bug, l'amélioration ou la question.
- Fournissez : version ciblée (ex. `snapshot_V2.4`), pas à pas pour reproduire et logs si possible.

2) Pull Requests
- Forkez le dépôt et créez une branche claire (`fix/nom`, `feat/nom`).
- Ouvrez une PR vers la branche `main` du dépôt d'origine.
- Décrivez le changement, pourquoi, et les fichiers modifiés.

3) Style de code
- Respectez la mise en forme existante du code.
- Cible : compatibilité Arduino/ESP32.

4) Tests
- Testez les changements localement sur un ESP32 si possible.
- Fournissez des étapes de test dans la PR.

5) Licence et utilisation
- Ce projet est licencié sous `CC BY-NC 4.0` (voir `LICENSE`). Cela interdit l'utilisation commerciale sans autorisation explicite du titulaire du copyright (LEGRIX b).

Remarques
- Si vous proposez du code critique (sécurité, gestion WiFi, etc.), documentez les risques et les tests effectués.
- Pour toute question, ouvrez une issue et mentionnez `@LEGRIX b` si nécessaire.
